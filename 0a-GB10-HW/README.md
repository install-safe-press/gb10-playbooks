Dell Pro Max GB10/Nvidia DGX Spark 等型態的設備只有Type C接口,機尾左側第一孔輸入一定是電源供應
初始設定的時螢幕鍵盤滑鼠連接有兩種方式
方式1.使用PC/NB透過GB10在初始狀態所生成的無線網路熱點訊號行連接,再設定連上環境下的wifi無線網路
方式2.將螢幕HDMI,一般鍵盤滑鼠透過Type C 集線器轉接到GB10 , 並接上有線網路進行初始化動作,需連上Internet網路時間大約十至二十分鐘。

詳閱初始設定指南
https://www.dell.com/support/kbdoc/zh-tw/000398800/dell-pro-max-with-gb-10-fcm1253-initial-setup-out-of-the-box-experience?lang=en

使用方式1需確保無線網路狀態定,如果狀態不穩定會出現請使用方式2連線

以下說明手邊可以拿到使用的Type C 集線器分別是<br>
Dell Pro 擴充基座 – WD25 正常  <br>
Dell DA200方型旅行用集線器-不匹配無法使用<br>
Dell DA300圓型旅行用集線器-正常<br>
無法識別型號台製USB HUB 外加Type C轉接頭<br>
以上透過轉接有線/無線 鍵盤滑鼠連接測試 <br>
